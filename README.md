# Promt-a-thon (AMD Slingshot)

- This repository documents my participation in the AMD Singshot Prompt-a-thon, held in Pune. The challenge was centered around building an innovative solution that could help individuals make better food choices and develop healthier eating habits by leveraging available data, user behavior, and contextual inputs.

- The competition provided a unique opportunity to combine creativity with technical problem-solving. The core idea was to design a system that not only recommends healthier alternatives but also adapts dynamically to user preferences, lifestyle, and environment. My solution focuses on bridging the gap between nutritional awareness and actionable decision-making, ensuring that users are guided toward sustainable eating habits rather than short-term fixes.
  
- I built this project entirely through my own coding efforts, embracing what I call my “vibe coding” approach—a mix of intuition, experimentation, and structured problem-solving. To power the solution, I utilized Google AntiGravity, which enabled me to integrate data-driven insights with a responsive interface. This allowed the system to process contextual information (such as time of day, activity level, or dietary restrictions) and provide tailored recommendations in real time.
  
The project emphasizes three key aspects:

  - Personalization – Every individual has unique dietary needs and preferences. My solution adapts to these differences, ensuring relevance and practicality.
  - Data Utilization – By leveraging available datasets and behavioral inputs, the system can make informed suggestions rather than generic advice.
  - Habit Formation – The goal is not just to suggest healthier meals but to encourage long-term behavioral change through consistent, positive reinforcement.
    
- Participating in the Prompt-a-thon was both challenging and rewarding. It pushed me to think critically about how technology can influence everyday decisions and improve quality of life. This repository serves as a record of my journey—capturing the design process, implementation details, and the lessons learned along the way.
  
- Ultimately, this project reflects my belief that technology should empower individuals to make smarter, healthier choices. By combining data, context, and user-centric design, we can create solutions that truly make a difference in people’s lives.
